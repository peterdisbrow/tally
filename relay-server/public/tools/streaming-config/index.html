6 Installation

    
Copy the file to:

    

      
macOS

      /Library/Application Support/Blackmagic Design/Switchers/Streaming.xml
    

    

      
Windows

      C:\ProgramData\Blackmagic Design\Switchers\Streaming.xml
    

    
Steps

    
    
      
  1. Close ATEM Software Control if it's open.
    2. 
      
  2. Copy the downloaded Streaming.xml file to the path above (you may need admin/root permissions).
    3. 
      
  3. Reopen ATEM Software Control.
    4. 
      
  4. Go to the Streaming palette and select your new service from the dropdown.
    5. 
      
  5. Enter your stream key in the key field.
    6. 
      
  6. Start streaming!
    7.
