Companion Integration Guide

Tally connects to Bitfocus Companion to read device status and trigger buttons across 300+ supported modules.

Quick Setup (5 minutes)
  1. Install Companion from bitfocus.io on the same computer as Tally
  2. Open Companion and add your devices (ATEM, OBS, ProPresenter, etc.)
  3. Create buttons for common actions (Start Stream, Camera 1-4, etc.)
  4. Tally auto-detects Companion on port 8888 — no extra config needed
  5. Say "press Start Stream" in the Tally chat to trigger buttons by name
Recommended Buttons for Church Production
What Tally Can Do With Companion
  • Trigger any button by name from chat, Telegram, or AutoPilot rules
  • Read live device status through Companion variables (program source, scene, timecodes)
  • Monitor all connections and alert if a device goes offline
  • Set custom variables for bidirectional Tally ↔ Companion workflows
  • Auto-watch common variables for ATEM, OBS, HyperDeck, ProPresenter, and more
